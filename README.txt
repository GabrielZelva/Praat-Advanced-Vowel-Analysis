-----------------------------------------------------------
Intro
-----------------------------------------------------------
This is a script designed for vowel analysis in Praat.
It can take in a whole corpus at once. 
-----------------------------------------------------------

-----------------------------------------------------------
How to use:
-----------------------------------------------------------

	- Open Praat
	- Click the "Praat" button in the upper menu 
			> Open Praat script 
				> Select the script
	
	- Click the "Run" button in the upper menu
	- Follow the instructions provided by the script
-----------------------------------------------------------
	#########################################
	  IF YOU ARE PHOTOSENSITIVE, BE CAREFUL
	PRAAT WILL FLICKER A LOT AND I MEAN A LOT
	#########################################
-----------------------------------------------------------
Additional instructions:
-----------------------------------------------------------
You should have 2 folders somewhere on your PC.

- The first one should contain the corpus, that is, .wav and .TextGrid files with corresponding names.
- The second one should be ideally empty and ready to be filled with the results generated by this script. 

Or you may just do it all in a single folder... 
For me it's a bit too chaotic, but if that is what you want, just input the same path twice. 
The script will not stand in your way. 
-----------------------------------------------------------

-----------------------------------------------------------
Example of use:
-----------------------------------------------------------
You may try it with the example audio + TextGrid pair attached in this repository.
For a bit of context, it is a young Spanish woman reading in an anechoic chamber.
Her dialect employs 5 vowels /a/, /e/, /i/, /o/ and /u/.
The segmentation was done automatically using MFA, so don't expect human level precision,
however, it is perfectly serviceable for this example. 
If you want to simulate the script analysing a whole corpus,
just copy and paste the pair a few times in the same folder
